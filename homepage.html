Página inicial do sistema

Home

Este sistema foi organizado em quatro áreas principais: Estoque, Lista, Compra e Consumo.

Clique em uma das opções abaixo para acessar a página correspondente.

Estoque

Mostra todos os itens cadastrados e a quantidade disponível de cada um.

Lista

Exibe os itens cadastrados para você marcar manualmente quais deseja comprar.

Consumo

Será usada para acompanhar a média de consumo dos itens ao longo do tempo.

Compra

Registra o que foi comprado de fato e calcula o valor total da compra.

Fluxo de uso

  1. Consultar os itens cadastrados no estoque.
  2. Marcar manualmente o que deseja comprar na lista.
  3. Observar com destaque os itens que estiverem com quantidade 0.
  4. Consultar os itens cadastrados no estoque.
  5. Usar a lista como apoio para o que está faltando.
  6. Registrar a compra real na página de compra.
  7. Usar a página de consumo quando essa parte for implementada.
