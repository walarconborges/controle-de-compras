Modelo inicial da página principal

Home

Este sistema foi organizado em três áreas principais: Estoque, Lista de Compras e Consumo Médio.

A navegação funciona pelas abas fixadas no rodapé da página.

Estoque

Mostra todos os itens cadastrados e a quantidade disponível de cada um.

Lista de Compras

Exibe os itens cadastrados para você marcar manualmente quais deseja comprar.

Consumo Médio

Será usada para acompanhar a média de consumo dos itens ao longo do tempo.

Fluxo de uso

  1. Consultar os itens cadastrados no estoque.
  2. Marcar manualmente o que deseja comprar na lista de compras.
  3. Observar com destaque os itens que estiverem com quantidade 0.
  4. Usar a aba de consumo médio quando essa parte for implementada.
